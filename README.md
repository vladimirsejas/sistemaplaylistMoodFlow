# sistemaplaylistMoodFlow
O MoodFlow é um sistema inteligente de playlists desenvolvido em Python com foco em Estrutura de Dados e integração com a API do Spotify. O projeto foi inspirado no trabalho acadêmico `projeto2sistemaplaylist`, criado pelo professor Orlando, funcionando como uma evolução moderna da proposta original. Enquanto o projeto inicial utilizava músicas simuladas para o estudo de estruturas encadeadas, o MoodFlow amplia o conceito utilizando músicas reais, BPM real e reprodução real através do Spotify.
O sistema foi desenvolvido com objetivo acadêmico e arquitetural, buscando unir conceitos clássicos de Estrutura de Dados com tecnologias modernas utilizadas em aplicações reais. O projeto mantém o foco no aprendizado de Lista Encadeada, Filas FIFO, modularização e separação de responsabilidades, porém agora integrado com APIs reais e controle de reprodução musical.
Cada música do sistema é representada por um objeto da classe `Musica`, armazenando informações como título, artista, álbum, BPM, duração, Spotify URI e Spotify URL.
A biblioteca principal será construída utilizando Lista Encadeada Simples implementada manualmente, sem uso de listas prontas do Python. Cada nodo da estrutura armazenará um objeto `Musica` e um ponteiro para o próximo nodo da lista encadeada.
O MoodFlow permitirá buscar músicas reais utilizando a API do Spotify, adicionar músicas na biblioteca, organizar playlists automaticamente através do BPM, controlar filas inteligentes por humor, reproduzir músicas reais e armazenar histórico de reprodução.
O sistema trabalhará inicialmente com três categorias de humor. Relaxar para músicas mais suaves e lentas. Focar para músicas moderadas voltadas à concentração. Animar para músicas mais intensas e agitadas.
A reprodução musical será realizada utilizando a biblioteca Spotipy e a Spotify Web API, permitindo controlar o Spotify Desktop, Spotify Web e dispositivos móveis através de playback real.
O projeto possui arquitetura modular organizada em componentes específicos responsáveis pela biblioteca encadeada, filas FIFO, integração com Spotify, controle de reprodução e interface do usuário.
O principal objetivo do MoodFlow é demonstrar como conceitos clássicos de Estrutura de Dados podem ser aplicados em sistemas modernos próximos de aplicações reais utilizadas atualmente.
